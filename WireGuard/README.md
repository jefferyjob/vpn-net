# WireGuard VPN VPN

因该客户端需要在 App Store 中下载，但是大陆市场不提供此app，所以不推荐部署此协议的vpn。

## 搭建步骤

### Shell搭建

```bash
wget https://get.vpnsetup.net/wg -O wireguard.sh
sudo bash wireguard.sh
```

### 客户端下载

- 下载地址：https://www.wireguard.com/install
