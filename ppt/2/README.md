# 方法二演示

## 基础环境

**服务器系统**

- CentOS 6

## 安装

### 脚本下载

```bash
wget https://raw.github.com/viljoviitanen/setup-simple-pptp-vpn/master/setup.sh
```

### 进行安装

```bash
sudo sh setup.sh
```

**执行完后有1、2、3种英文，对应如下中文意思。**  

1. 安装vpn服务  
2. 修复vpn  
3. 添加vpn用户  

我们要安装PPTP vpn，输入1，然后确认。
等待安装完成，就会出现账号和密码啦，默认账号是vpn，密码是一串随机数字，现在你已经搭建 pptp vpn 成功了
